Id Layout#

Defined in package com.scandit.datacapture.id.ui.overlay

IdLayout#

Added in version 6.6.0

A kind of IdCaptureOverlay UI displayed to assist a user in scanning specific types of personal identification documents.

AUTO#

Added in version 6.6.0

The UI determined automatically, based on a set of IdCaptureSettings.supportedDocuments.

NONE#

Added in version 6.6.0

No UI displayed.

TD1#

Added in version 6.6.0

The UI suitable for scanning Size 1 Machine Readable Official Travel Documents (TD1) or Swiss Driver Licenses.

TD2#

Added in version 6.6.0

The UI suitable for scanning Size 2 Machine Readable Official Travel Documents (TD2) or Format B Machine Readable Visas (MRV-B).

TD3#

Added in version 6.6.0

The UI suitable for scanning Size 3 Machine Readable Official Travel Documents (TD3).

MRV_A#

Added in version 6.6.0

The UI suitable for scanning Format A Machine Readable Visas (MRV-A).

VIZ#

Added in version 6.6.0

The UI suitable for scanning documents or their parts intended to be read by humans (for example Visual Inspection Zone (VIZ) of a Machine Readable Travel Document (MRTD)).

PDF417#

Added in version 6.6.0

The UI suitable for scanning PDF417 from personal identification documents.
