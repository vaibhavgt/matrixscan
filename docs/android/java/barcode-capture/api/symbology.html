Symbology#

Defined in package com.scandit.datacapture.barcode.data

The Scandit Data Capture SDK is capable of reading many different types of barcodes. The type of a barcode is referred to as its symbology. Below is a list of all symbologies supported by the Scandit Data Capture SDK. See also Barcode Symbologies for more details about each symbology and Configure Which Barcodes Are Read to understand all the configuration options that are available.

Symbology#

Added in version 6.0.0

An enumeration of barcode types (symbologies) supported by the Scandit Data Capture SDK.

The availability of the symbology depends on the licensed features.
